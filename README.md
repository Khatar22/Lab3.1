# Сельскохозяйственный симулятор на JS (ООП, модули)

## Запуск локально
1. Откройте `index.html` в браузере.
2. Никаких сборок не требуется: ES-модули подключены напрямую.

## Структура
- `src/classes/*` — базовые классы и наследники.
- `src/grid.js` — сетка, расчет влажности по расстоянию к воде.
- `src/tools.js` — инструменты: лопата, ведро, семена.
- `src/main.js` — инициализация и игровой цикл.
- `styles/styles.css` — визуализация сетки и растений.
- `assets/*` — изображения культур и воды.

## Проверка линтером
- Установите ESLint/Prettier глобально или локально.
- Запустите:
  - `npx eslint src --ext .js`
  - `npx prettier --check .`
